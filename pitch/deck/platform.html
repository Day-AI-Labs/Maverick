DAYBREAK LABSThe chokepoint04 / 16
Capability & policy

Least privilege, enforced.

Capability tokens

Every specialist gets a token scoped to exactly what it needs. Attenuate-only — a token can be narrowed, never widened, as it passes down the swarm.

Policy engine

Each action is evaluated against policy: allow · deny · require-human. The decision is recorded with the action.

Egress lock

Data can't leave unless policy says so. A successful prompt injection still can't move data out — the lock sits below the agent.

Privilege is bound to the action, not the prompt. The agent can be wrong and the boundary still holds.

The audit

Every action, signed and hash-chained.

  • Tamper-evident. Each entry is hash-chained to the one before it; altering a row breaks the chain.
  • Verifiable offline. No call home — a regulator can check the chain in an air-gapped room.
  • The Operating Record. One signed ledger of what the workforce did, decided, and refused.

DAYBREAK LABSThe audit06 / 16
Budgets & sandbox

Bounded blast radius.

Hard budget caps — enforced at record time
  • Tokens. A run can't burn past its token ceiling.
  • Dollars. A spend cap stops the work, not the invoice.
  • Wall-clock. Time-boxed — a runaway loop is capped, not waited out.
  • Tools. A cap on tool calls bounds the action count.

Sandbox

Network-off by default. Tools run in a sandbox with no egress unless policy grants it. All shell routes through sandbox.exec() — nothing runs around it.

A specialist that goes wrong is contained: capped budget, no network, no privilege it wasn't handed.

The golden path

Watch it refuse.

A seeded scenario you run live. The platform refuses the dangerous action and leaves a tamper-evident receipt for every step.

golden-path receipts
boot finance specialist  SEALED
$60k wire          DENY
$6k release       REQUIRE_HUMAN
runaway loop      CAPPED
alter audited row  CAUGHT
every row tamper-evident
  • SEALED. Boots with no shell, no payments — least privilege from the start.
  • DENY. A $60k wire is blocked by policy.
  • REQUIRE_HUMAN. A $6k release escalates to a person.
  • CAPPED. A runaway loop hits the budget and stops.
  • CAUGHT. Altering one audited row breaks the chain.
