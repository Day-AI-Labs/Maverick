Couldn’t reach the dashboard

Maverick tries to start the dashboard for you. If it never comes up, the maverick command usually isn’t installed yet, or isn’t on the PATH this app can see.

  1. Make sure Maverick is installed (see the setup guide).
  2. Open a terminal and run maverick dashboard — this window connects automatically once it’s up.
