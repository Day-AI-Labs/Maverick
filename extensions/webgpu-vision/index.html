WebGPU local vision primitives

Local only. Images you pick here are processed entirely inside this page — on your GPU/CPU — and are never uploaded anywhere. This page makes no network requests.

Honest scope: hand-written image ops (grayscale, Sobel edge map) running as real WebGPU compute shaders, plus an 8x8 average-hash for "are these two screenshots the same screen?" comparisons. Not a trained vision model — nothing here recognises objects or reads text.
