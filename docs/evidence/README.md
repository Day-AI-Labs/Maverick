# Evidence artifacts

Machine-generated proofs that a load-bearing architectural guarantee actually
holds on this codebase — the kind of artifact a security/architecture reviewer
or an auditor asks for ("show me it runs out-of-process", not "tell me it does").

| Artifact | Generated by | What it proves |
| --- | --- | --- |
| `control-data-plane-e2e.json` | `python -m maverick.control_data_plane_e2e --out <path>` | A goal submitted through the real `QueueDispatcher` is executed by a *separate* `Worker` against the shared stores — never on the control path — and its terminal status flows back through the shared world DB. Agent execution is stubbed at the LLM boundary; the harness proves the dispatch plumbing, not the agent loop. |

These are regenerated on demand and verified on every CI run (each generator
exposes a `--ci` flag that exits non-zero if the guarantee fails). The committed
copies are reference samples; the `at` timestamps move when regenerated.
