# What this skill does

Plans a public-relations push around a specific announcement, milestone, or campaign. Produces a PR plan with story angles, prioritized media/influencer targets, key messages, and a dated timeline from embargo through follow-up.

# Steps

1. Establish the news hook and objective via knowledge_search: what is being announced, the audience, the business goal, available spokespeople, and any embargo or launch date. If the hook is weak or undated, surface that before planning — a plan without real news fails.
2. Derive 2-4 distinct story angles tied to the audience (e.g., customer impact, market trend, executive POV). Map each to a target outlet type and craft the core message and proof point per angle, grounded in approved facts.
3. Build the target list from knowledge_search of relevant beats/outlets/analysts; prioritize by fit and reach, and note the angle each target gets. Mark any target as a best-guess if no contact/beat data is confirmed.
4. Sequence a timeline: prep (assets, briefings, embargo offers), launch day, and follow-up (bylines, social, amplification). Assign owners and dependencies.
5. Report the plan as angles + target table + timeline, listing assumptions (date, spokesperson availability, asset readiness) and what must be confirmed before outreach begins.

# Notes

The plan is wrong if angles aren't differentiated, if targets don't map to a real beat, or if the timeline ignores the embargo. Do not state contacts or commitments as confirmed unless sourced; mark them tentative. Pitching, embargo offers, and outreach are drafted for a human PR lead to execute — this skill does not send anything. Skip when there is no genuine news; manufactured hooks damage media relationships.
