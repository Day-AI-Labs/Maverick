# What this skill does

Drafts a periodic investor update: highlights, lowlights, core metrics, the ask, and a candid line on what's hard — the format that keeps investors helpful between rounds.

# Steps

1. Lead with TL;DR and the explicit ask (intros, hires, advice). Investors skim; put the action first.
2. Report the core metrics consistently period over period from `spreadsheet`: revenue/ARR, growth, burn, runway, and one or two north-star numbers. Never quietly change definitions.
3. Give highlights AND lowlights honestly — a missing lowlights section reads as spin and erodes trust.
4. Add asks, team changes, and a short 'how you can help'. Keep it scannable; detail goes to an appendix. State the reporting basis and hand off for sign-off.

# Notes

Updates lose their value when metrics shift definitions silently, lowlights vanish, or runway math is stale. Financial figures going to investors need a human owner's sign-off before sending — this skill drafts, it does not send.
