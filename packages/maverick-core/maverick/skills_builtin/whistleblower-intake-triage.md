# What this skill does

Triages a single inbound whistleblower / ethics-hotline report and produces a draft intake record: issue categorization, severity and urgency rating, jurisdiction/policy mapping, conflict screen, and a recommended routing and acknowledgment plan. Output prepares the case for a case manager or ethics officer to formally open and assign; it is not an investigation or a finding.

# Steps

1. Capture the report intake fields exactly as submitted: allegation summary, named subjects, dates, location/entity, reporter status (named vs. anonymous), and any evidence referenced. Preserve the reporter's words for the core allegation; do not editorialize or assume guilt — these are allegations, not facts.
2. Use `knowledge_search` to map the allegation to the case taxonomy and governing policy/regulation (e.g., fraud, harassment, safety, financial-reporting, data-privacy, retaliation) and to the applicable whistleblower-protection and confidentiality rules. Cite the policy/category codes and any mandatory-reporting or regulatory-notification trigger the allegation type carries.
3. Rate severity and urgency (e.g., imminent safety/legal/financial-statement risk = expedite) and run a conflict/recusal screen: identify whether any named subject sits in the normal investigation chain, requiring routing to an independent path. Flag retaliation indicators and protected-reporter status.
4. Recommend routing (owning function, investigator pool, escalation tier), the acknowledgment/confidentiality handling for the reporter, and any external/regulatory notification clock that has started. Report the draft intake record with category, severity, routing, and assumptions, and hand off to the case manager to open and assign; do not contact subjects.

# Notes

This is triage, not adjudication: never treat an allegation as substantiated, and never weaken reporter confidentiality or anonymity — a breach can itself be unlawful retaliation. The highest-risk error is mis-rating severity (burying an imminent harm) or routing a case into a chain that includes the subject; the conflict screen exists to prevent the latter and must not be skipped. Some categories (safety, financial-reporting, certain crimes) carry mandatory regulatory clocks — flag them even when uncertain so a human can confirm. The triage drafts and recommends; opening the case, notifying regulators, and interviewing subjects are human decisions. Do not use to communicate with the reporter or any subject, or to make retaliation findings.
