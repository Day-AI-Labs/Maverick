# What this skill does

Builds and audits a diligence data-room index: the document checklist by workstream, what's present vs missing, and the gaps that will stall a deal.

# Steps

1. Lay out the checklist by workstream: corporate, financial, legal/contracts, IP, HR, tax, compliance, IT/security, and commercial.
2. Map each required document to present / partial / missing, with the responsible owner and a target date for gaps.
3. Flag the red-line items that block or reprice a deal: missing IP assignments, change-of-control clauses, unsigned key contracts, open litigation, tax exposures.
4. Produce a gap report ranked by deal impact, plus a clean folder index for the data room. State assumptions and hand off.

# Notes

Diligence stalls on change-of-control clauses, missing IP assignments, and stale financials. Surface those early. This skill organizes and flags; legal and financial conclusions are for the deal team and advisors.
